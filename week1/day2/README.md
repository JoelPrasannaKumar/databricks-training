# Day 2 - SQL Joins Practice

This folder contains SQL JOIN practice queries and assignments using PostgreSQL.

## Files
- `datacreation.sql` – creates tables and inserts sample data
- `sql practice queries.pdf` – SQL practice questions
- `sql answers 36 to 55.sql` – answers for questions 36 to 55
- `sql_joins_assignment.pdf` – SQL joins assignment questions
- `sql joins assignment answers.sql` – solutions for the joins assignment
- `sql_joins_starter.sql` – starter file with schema and sample data
- `outputs/` – query outputs and practice results

## Topics Covered
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN
- JOIN with GROUP BY
- JOIN with aggregate functions
- Nested and correlated queries
- Handling NULL values in joins

## Tables Used
- Students
- Courses
- Enrollments
- Instructors
- Employee
- Department
- Project

## Concepts Practiced
- Finding unmatched rows
- Combining data from multiple tables
- Identifying missing relationships
- Counting related records
- Using aliases with joins
- Writing reports using multiple joins

## How to Run
1. Execute the starter/data creation SQL file first.
2. Run the practice queries one by one.
3. Compare outputs with the answers file.
4. Modify joins and conditions for additional practice.

## Database
Queries are mainly written for:
- PostgreSQL
- MySQL compatible syntax in some practice files
