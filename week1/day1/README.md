# Day 1 - SQL Practice

This folder contains the SQL practice materials for Day 1.

## Files
- `datacreation.sql` – SQL script to create and insert the sample tables/data
- `sql practice queries.pdf` – list of SQL practice questions
- `sql answers 1 to 35.sql` – answers for questions 1 to 35

## Topics Covered
- Basic SELECT queries
- WHERE conditions
- LIKE pattern matching
- Date queries
- Aggregate functions
- GROUP BY and HAVING
- ORDER BY

## Notes
- The queries are based on the Employee, Department, and Project tables.
- The answers are written mainly in MySQL syntax.

## How to Use
1. Run `datacreation.sql` first to create the tables and insert sample data.
2. Refer to `sql practice queries.pdf` for the questions.
3. Use `sql answers 1 to 35.sql` to practice and verify your results.
